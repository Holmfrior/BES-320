# DD-AI: A New Way to Learn Discourse Analysis

DD-AI is an interactive educational platform that teaches users how to identify and map relationships between ideas in a text using Rhetorical Structure Theory (RST).


## Definition
Rhetorical Structure Theory (RST) is a framework used to analyze the organization of natural text by identifying the relationships between its different parts. It focuses on how a text functions as a coherent unit.
